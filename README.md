# GamePlan

GamePlan is a full-stack platform designed to manage and streamline operations for competitive cheer programs.

This repository is structured as a monorepo containing both the frontend and backend applications.

---

## Project Structure

```
GamePlan/
├── frontend/
└── backend/
```

---

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- REST API integration

### Backend
- Node.js
- Express
- TypeScript
- PostgreSQL
- Prisma ORM
- JWT Authentication
- Role-Based Access Control (RBAC)

---

## Getting Started

### Clone the repository

git clone https://github.com/amacthedev/GamePlan.git  
cd GamePlan  

### Install dependencies

Backend:

cd backend  
npm install  

Frontend:

cd ../frontend  
npm install  

---

## Running the Applications

Start backend:

cd backend  
npm run dev  

Start frontend:

cd frontend  
npm run dev  

---

## Development Workflow

- Create feature branches from main  
- Open pull requests for changes  
- CI checks must pass before merging
